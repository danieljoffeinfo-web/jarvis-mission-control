# Jarvis Mission Control

Black-and-white mission control dashboard for prompting agents, tracking goals, and monitoring mission progress.

Features:
- Directive console for dispatching prompts
- Fleet overview with agent workload and status
- Goal board with completion toggles
- Milestone telemetry and activity feed
- Local persistence via browser localStorage

Stack:
- Static HTML
- CSS
- Vanilla JavaScript
- Vercel deployment ready
